## PLAY STORE SETUP STEPS

1. Go to [play.google.com/console](https://play.google.com/console) → Create app
2. App name: **ZenSpace**
3. Default language: **English (India)**
4. App or game: **App**
5. Free or paid: **Free** (with in-app purchases)
6. Upload APK/AAB: Build with `eas build --platform android` using Expo EAS
7. Fill in Store listing with the text above
8. Upload feature graphic + icon + screenshots
9. Set Privacy Policy URL
10. Complete Content Rating questionnaire
11. Set up in-app products (subscriptions) in the Monetize section
12. Submit for review (~3–7 business days)
